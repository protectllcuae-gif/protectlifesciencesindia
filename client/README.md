# Protect Life Sciences Client

This is the React frontend for Protect Life Sciences.

## Structure

- `src/pages/` — route pages
- `src/components/` — reusable UI components
- `src/assets/` — shared styling and assets
- `public/` — static files and images

## Run

```bash
cd client
npm install
npm run dev
```

## Build

```bash
npm run build
```
